Features

Everything between
discovery and running.

GitStore handles runtime detection, dependency installation, and process management so you don't have to.

Smart Discovery

Curated collections from awesome-lists, GitHub trending, and community suggestions. Not just a list — a catalog.

One-Click Install

Click GET. GitStore clones the repo, detects the runtime, installs dependencies, and builds. You wait 30 seconds, not 30 minutes.

Auto Runtime Detection

Node.js, Python, Rust, Go, Docker — automatically detected from the repo's manifest files. No manual configuration.

GitHub Trending

See what's blowing up on GitHub right now. Daily, weekly, monthly — filtered by language. Install trending repos before everyone else.

Risk Assessment

Every repo gets a safety score based on stars, age, license, and suspicious patterns. Know what you're running before you run it.

AI-Native (MCP)

Built-in MCP server. Ask Claude to "find me a markdown editor and install it" and it just works. Your AI assistant manages your apps.

How It Works

Three clicks to running.

From GitHub URL to running application in under a minute.

1

Discover

Browse 800+ curated apps across 13 categories. Check trending repos. Read the README right in the app. See risk scores and requirements.

2

Install

Click GET. GitStore detects the runtime (Node, Python, Rust, Go, Docker), clones the repo, installs dependencies, and builds — automatically.

3

Run

Hit Start. Your app launches on localhost with the right ports and environment. Stop, restart, update, or uninstall from one dashboard.

AI-Native

install your apps.
